# Project jmh-tests
This project features JMH tests.

## Building, Running and Checking

1. Navigate to the project directory:
```
cd <projects_directory>/jmh-tests
```

2. Build the project:
```
mvn clean install
```

3. Run the server:
```
java -jar ./target/jmh-tests-1.0.0-SNAPSHOT.jar
```
